The Server

The websocket server will create our port and listen for clients. Specifically, the server will listen for incoming messages from the Draw Client and publish update messages for the Graphics Client to consume.

1.In your project directory, create a new Processing File ‘Server’

2.Import websockets at the top of your file, and create a WebsocketServer class named “ws’

import websockets.*;

WebsocketServer ws;

3.Setup our main environment; screen-size and server port import websockets.*;

WebsocketServer ws;

void setup(){ size(800, 800);

ws= new WebsocketServer(this,8025,"/points");

}

This will create a single environment, set the screen size to 800 x 800 pixels, and create a new Websocket Server object. Set the port to 8025, with an endpoint named ‘server’, where our client and server will ‘handshake’. Note ‘Objects’ are used as the primary way to instance classes in Processing.

4.Draw our server screen. We won’t be displaying anything here yet, but it’s a helpful indicator that everything is work appropriately.

import websockets.*;

WebsocketServer ws;

void setup(){ size(800, 800);

ws= new WebsocketServer(this,8025,"/points");

}

void draw(){ background(0);

}

5. Get incoming request messages and post them to main server port.

import websockets.*;

WebsocketServer ws;

void setup(){ size(800, 800);

ws = new WebsocketServer(this,8025,"/points");

}

void draw(){ background(0);

}

void webSocketServerEvent(String msg){ println(msg); ws.sendMessage(msg);

}

Here we listen for server events (messages), print them to the console and post them using the sendMessage() method for the Graphics Client to consume.

The Draw Client

When we’re done our DrawClient will be able to communicate with the Server and Graphics Clients from any PC on the same network.

The draw client is setup similarly to the Server.

import websockets.*;

WebsocketClient socket;

void setup(){ size(800, 800);

socket = new WebsocketClient(this, "ws://localhost:8025/points");

}

void draw(){

if (mousePressed == true) { stroke(255);

line(pmouseX, pmouseY, mouseX, mouseY);

socket.sendMessage(pmouseX + " " + pmouseY + " " + mouseX + " " + mouseY);

}

}

void webSocketServerEvent(String drw_msg){ println(drw_msg);

}

1. Instead of the Server, this time we instance the websocket client class, and set the port.

socket = new WebsocketClient(this, "ws://localhost:8025/points");

2.If the mouse is pressed; draw a new line starting with the mouse pressed x-y position, ending with the mouse released x-y position, the share those positions with the Server.

stroke(255);

line(pmouseX, pmouseY, mouseX, mouseY);

socket.sendMessage(pmouseX + " " + pmouseY + " " + mouseX + " " + mouseY);

The Graphics Client

The graphics client will use data received from the Draw Client/Server to control elements of our game world in real-time.

1.In Godot, create a scene, a script on the root node and a child HTTP Node

2.On ready instance a new WebSocketClient and connect the following three signals to the instance.

extends Node

var ws = null

func _ready():

ws = WebSocketClient.new()

ws.connect("connection_established", self, "_connection_established") ws.connect("connection_closed", self, "_connection_closed") ws.connect("connection_error", self, "_connection_error")

3. Add the port url

var url = "ws://127.0.0.1:8025/points"

var ws = null
